[vue-cli](https://cli.vuejs.org/config/)

## 构建方法

1. 安装 node
2. `npm install` 安装依赖
3. `npm run build` 构建项目

## 部署

将生成的 `dist` 目录下的静态文件放到服务器静态文件根目录, 
配置指向 `index.html` 文件路径.

输出目录可以到 `vue.config.js` 里面修改.
